⚠️ Chú ý (Dễ bị hỏi thi):

Kiến thức IT Tuần 1-2: Khái niệm API

Application Programming Interface (Giao diện lập trình ứng dụng)

💡 Liên tưởng thực tế: API giống như người phục vụ trong nhà hàng. Bạn (Client) xem menu và gọi món. Người phục vụ (API) mang yêu cầu của bạn vào bếp (Server/Database), sau đó bưng món ăn (Dữ liệu) ra cho bạn. Bạn hoàn toàn không cần biết đầu bếp nấu như thế nào.
